# Session Wrapup

Use when the current topic is ending and follow-up is uncertain.

## Workflow

1. Write a short note to `work/session-end/<timestamp>-wrapup.md`.
2. Keep the note focused on what changed, what was decided, and what should not be repeated.
3. Update `work/session-end/LATEST.md`.
4. Keep the note short enough that a fresh session can read it in under a minute.

## Note shape

- goal
- result
- key decisions
- mistakes to avoid
- reusable lessons
- unfinished items
